# SSSALU Website

South Sudanese Student Association at African Leadership University official website.

## Project Structure

```
sssalu/
└── frontend/          # React Frontend
    ├── src/           # React components & pages
    ├── public/        # Static assets
    └── package.json   # Node dependencies
```

## Setup & Installation

### Frontend (React + TypeScript)
```bash
cd frontend
npm install
npm start
```

## Features

- **Responsive Design**: Mobile-first approach with modern UI
- **Static Content**: Fast and efficient static website
- **Image Gallery**: Showcase events and community photos
- **Contact Information**: Easy access to association details

## Tech Stack

- **Frontend**: React 18, TypeScript, React Router
- **Styling**: CSS3 with CSS Variables, Responsive Grid/Flexbox

## Development

1. **Start Frontend**: `cd frontend && npm start`
2. **Access**: Frontend at http://localhost:3000

## Build for Production

```bash
cd frontend
npm run build
```
